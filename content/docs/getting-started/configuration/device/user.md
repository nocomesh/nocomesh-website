## Options

| Name                         | Example            | Notes                                                                           |
| ---------------------------- | ------------------ | ------------------------------------------------------------------------------- |
| Long Name                    | My Meshtastic Node | Descriptive name of the node                                                    |
| Short Name                   | MMN                | Short Identifier (typically 3-4 characters)                                     |
| Unmessageable                | False              | Only enable on nodes that you won't be monitoring, like solar CLIENT_BASE nodes |
| Licensed amateur radio (HAM) | False              | Indicates if the user is a licensed amateur radio operator                      |

### What should I name my device?

You can name your device whatever you'd like! Just try to avoid overlap on names with nodes you can see
